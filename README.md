# Web ODE Solver
Integrate models based on systems of differential equations (ODEs), fit model parameters based on (experimental) datasets, and perform sensitivity analyses of the model.

## Key features
* Import data that is plotted alongside the model graphs.
* Solve (integrate) systems of differential equations. Equations can be typed as text. Add/remove equations as needed.
* Define model constants and variables.
* Plot results.
* Fit parameters within systems of differential equations based on imported data.
* Perform sensitivity analysis on model inputs.

## Project setup
Front end:  
* React using VITE framework  
* Using Chart.js for plotting graphs  

Back end:  
* Maven project using Spring Boot Framework (Java)
